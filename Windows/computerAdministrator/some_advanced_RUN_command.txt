Win+R[cmd], Ctrl+Shift+Enter, Alt+Y => to open "CMD" as administrator

sc /scannow => Scans and repairs system files.

bootrec /fixmbr => fix Master Boot Record.

dism /online /cleanup-Image /restorehealth: Repair Windows image.

ipconfig /flushdns => Clear DNS cache.

chkdsk f C: => Check disk for errors.

chkdsk /r X => to repair corrupted memory (X: is memory drive)

cleanmgr => to run "Disk Cleanup Manager"

Win+R [sysdm.cpl ,4]: create restore point.
